Game Projects


GOAAAALL

[lua, LOVE2D]

This is a one-on-one football game wherein each player controls one character each. Each player bumps or kicks the ball to the opposing goal post to score a point and whoever scores more after 3 and a half minutes win. In case of a tie, there will be a sudden death, where the gameplay resumes from where it stopped as the timer went down to 0 and whoever scores first wins.


Third Person Shooter

[C#, UnityEngine]
Link to GitHub repository

This involves the core mechanics of a third person shooter, specifically player movement and look-around, both of which the camera follows, a weapon that shoots a projectile (like the red particles in the image) on command towards the target reticle, and objects--such as spheres in the image--that will be destroyed once it is shot a couple of times.


Tile-Based Movement

[C#, UnityEngine]
Link to GitHub repository

This involves the core mechanics of a game that utilizes tile-based movement. Other than the basic movement in this implementation, this also includes tiles that force movement to a specific direction (arrow tiles), tiles that force movement based on which direction the player is facing (blood-like tiles), and walls (such as the volcano).


Shoot 'Em Up

[C#, UnityEngine]
Link to GitHub repository

This involves the core mechanics of a shoot 'em up (or shmup), which includes the player being able to control a character (the rectangle) that shoots projectiles (green lines) to destroy the enemies (the circles). The player has an HP, shown by the bar below the character model, which they have to preserve. The enemies, on the other hand, come from one side of the screen, and they continue to move towards the other side without pause, unless they are destroyed. They also shoot projectiles (red lines) back at the player.


Angry 50: The Tri-shot Update

[lua, LOVE2D]

From the Harvard instructor's implementation of the core mechanics of Angry Birds, I added a new feature that allows the bird to spawn two more versions of itself above and below it, and the duplicates share the same velocity as the original one. In short, I added the mechanics of the small blue bird from the Angry Birds games.


Platformer

[C#, UnityEngine]
Link to GitHub repository

This involves the core mechanics of a platformer, which includes moving and stationary platforms, walls, items to be picked up (small white squares), and warps (pink squares) that teleport the player to a specific spot within the map. The camera also follows the player's movement; in my implementation, there is no camera drift.

Web Application Projects


laban.gg

[Python, Django, HTML]

This website, when deployed, is meant to be a centralized platform for Philippine-based tournaments for fighting games in which tournament organizers can host their tournaments, and attendees can participate or keep track of said tournaments. This web application will be able to aid organizers in marketing and administering their tournaments, and aid players to look for opportunities to build and cement their status in the esports scene.
